📝 cyt图片文件夹说明

请将cyt的照片重命名为以下格式并放入此文件夹：

支持的文件名格式：
✅ 1.jpg    ✅ 1.png    ✅ 1.jpeg   ✅ 1.gif    ✅ 1.webp
✅ 2.jpg    ✅ 2.png    ✅ 2.jpeg   ✅ 2.gif    ✅ 2.webp
✅ 3.jpg    ✅ 3.png    ✅ 3.jpeg   ✅ 3.gif    ✅ 3.webp
...最多到 10.jpg/10.png/10.jpeg/10.gif/10.webp

📋 使用步骤：
1. 选择cyt最好看的照片
2. 重命名为 1.jpg, 2.jpg, 3.jpg... 的格式
3. 复制到这个文件夹
4. 启动服务器，页面会自动加载并展示

🎨 照片展示效果：
- 照片会以悬挂相框的形式显示
- 分布在屏幕左右两侧，避开中心的3D蛋糕
- 每10秒轮换一批，循环播放所有照片
- 每张照片都会标注"cyt的美好时光"

💡 小贴士：
- 建议使用横向或正方形的照片效果更好
- 文件大小建议控制在5MB以内
- 照片越多，轮播效果越丰富！
